# BlackRoad Helm Charts

Official Helm charts for BlackRoad OS services.

## Usage

```bash
helm repo add blackroad https://blackroad-os.github.io/helm-charts
helm repo update
helm install my-agents blackroad/blackroad-agents
```

## Available Charts

| Chart | Description |
|-------|-------------|
| blackroad-agents | AI Agent Orchestration Platform |
| blackroad-api | API Gateway |
| blackroad-mesh | Distributed Computing Mesh |
